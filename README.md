# LaLiga-2023-2024-Season-Analysis
This project presents a detailed analysis of the 2023–2024 LaLiga season, with an additional focused analysis on Real Madrid’s performance during the same season. The goal of this analysis is to uncover key trends, performance indicators, and team/player contributions using data-driven methods.

## Data
Utilized data from: https://www.kaggle.com/datasets/whisperingkahuna/la-liga-202324-players-and-team-

## Analysis
### Section 1 - LaLiga Table 2023-2024
- 2023-2024 LaLiga Table: A bar chart visualizing the final standings of the season where its comparing the points earned by team. 
- LaLiga 2023-2024 Season: Wins, Draws, and Losses: A grouped bar graph comparing each teams wins,draws and losses. 
- 2023-2024 LaLiga Win Percentage: A bar chart comparing the win percentage of each team in the season.
- 2023-2024 LaLiga Goal Difference: A bar chart comparing the goal difference of each team in the season.

### Section 2 - LaLiga Table Home 2023-2024

- 2023-2024 LaLiga Home Win Percentage: A bar chart comparing the win percentage of each team in the season at home.
